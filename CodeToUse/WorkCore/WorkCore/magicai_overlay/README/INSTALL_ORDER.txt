MagicAI WorkCore Overlay
=========================

This overlay contains only the host-side files needed to register the cleaned
WorkCore source pack inside MagicAI.

Install order:
1. Copy config/workcore.php into MagicAI config/
2. Copy app/Providers/WorkCoreServiceProvider.php into MagicAI app/Providers/
3. Copy routes/workcore.php into MagicAI routes/
4. Register App\Providers\WorkCoreServiceProvider::class in config/app.php
5. Import one domain slice at a time using MAGICAI_PREMERGE/42_MAGICAI_IMPORT_ORDER.md
