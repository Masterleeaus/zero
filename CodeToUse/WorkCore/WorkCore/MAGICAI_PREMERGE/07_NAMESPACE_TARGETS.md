# Namespace Targets

Recommended host-side placement based on the guide.

- Controllers -> `App\Http\Controllers\WorkCore\...`
- Models -> `App\Models\WorkCore\...`
- Services -> `App\Services\WorkCore\...`
- Jobs -> `App\Jobs\WorkCore\...`
- Events/Listeners -> `App\Events\WorkCore`, `App\Listeners\WorkCore`
- Routes -> `routes/workcore.php`
- Views -> `resources/views/workcore/...`
- Config -> `config/workcore.php` only
