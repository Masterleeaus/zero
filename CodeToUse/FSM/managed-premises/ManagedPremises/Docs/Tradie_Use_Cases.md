# Tradie-focused use cases (ManagedPremises)

- **Cleaner**: room list + cleaning checklist templates per property.
- **Electrician**: access register + hazard controls + asset serials (e.g. switchboard photo, RCD details).
- **Plumber**: appliance asset list (HWS model/serial), hazards (confined space), service windows.
- **Builder**: handover checklist + photos + linked work orders.
