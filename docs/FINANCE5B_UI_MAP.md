# Finance Pass 5B UI Map

Views created under `resources/views/default/panel/user/money/`:
- `budgets/index.blade.php` — list + create budget form
- `budget-variance/index.blade.php` — company variance report with line-level risk
- `scenarios/index.blade.php` — scenario type selector + result display
- `recommendations/index.blade.php` — pending recommendation queue
- `recommendations/review.blade.php` — approve/reject/dismiss single recommendation

All views extend `panel.layout.app`.
