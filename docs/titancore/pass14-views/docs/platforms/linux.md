# Linux Platform Documentation

## Overview

Desktop Linux app using Electron.

## Setup

1. Install Node.js 20+ and Electron.
2. Run `npm install`.
3. Run `npm run dev:linux`.

## Features

- Native Linux UI (GTK)
- AppImage support
- Snap packaging

## Building

- Use `npm run build:linux`.
- Package for AppImage or Snap.

## Deployment

- Distribute via AppImage or package managers.

## Troubleshooting

- Ensure Linux development tools are installed.
- Check for distro-specific issues.
