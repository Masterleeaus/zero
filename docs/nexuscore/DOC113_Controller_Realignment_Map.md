# Controller Realignment Map

## Objective
Assign controllers to canonical Nexus modes.

Pattern:
App\Http\Controllers\<Mode>\<Entity>Controller

Examples:

Jobs
JobController
ChecklistController
EvidenceController

Comms
ConversationController
MessageController

Finance
InvoiceController
PaymentController

Admin
RoleController
PermissionController

Social
CampaignController
PostController
PlatformController
