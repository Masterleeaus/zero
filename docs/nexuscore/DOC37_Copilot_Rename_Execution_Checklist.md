# DOC 37 — Copilot Rename Execution Checklist

Before rename:
- full rescan complete
- mode ownership assigned
- ambiguous terms locked
- canonical path map approved

During rename:
- rename in place
- preserve UI structure
- rewrite all references
- do not scaffold replacement trees
- delete duplicates immediately

After rename:
- verify routes
- verify views
- verify translations
- verify services
- verify signal ownership
- regenerate docs
- emit mapping report
