# DOC 25 — AEGIS Interception Rules

AEGIS can intercept at any stage.

Primary interception reasons:
- permission mismatch
- compliance failure
- risk threshold exceeded
- invalid state transition
- policy conflict
- duplicate/conflicting record
- audit anomaly

AEGIS outcomes:
- approve
- hold
- reject
- escalate
- request revision

AEGIS never creates domain artifacts.
AEGIS governs and records.
