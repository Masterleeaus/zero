# DOC 33 — Ambiguous Term Lock List

These terms must be explicitly locked before full conversion:

- Platform
- Account
- Campaign
- Post
- Program
- Workflow
- Contact
- Channel
- Draft
- Agent
- Command
- Inbox
- Schedule
- Analytics
- Insights

Each ambiguous term needs:
1. universal meaning
2. mode-specific visible meaning
3. file/path implications
4. route/controller implications
