# DOC 24 — Sentinel Validation Checkpoints

Each Sentinel validates readiness before processing.

Jobs Sentinel:
- customer/contact bound
- schedule valid
- service scope defined
- checklist/SOP attached if required

Comms Sentinel:
- contact/channel bound
- message target valid
- permissions valid
- delivery path available

Finance Sentinel:
- ledger/contact bound
- amount valid
- tax/payment fields valid
- approval chain attached when needed

Admin Sentinel:
- department/policy context valid
- permissions valid
- compliance requirements attached

Social Sentinel:
- campaign/account/platform valid
- schedule valid
- media/content valid
- publish permissions valid
