# Source to Mode Mapping

## High-Level Mapping

Legacy Social/Content Surfaces
- posts -> Social Mode OR Jobs/Comms/Finance draft surfaces depending ownership
- campaigns -> Social Mode OR Jobs programs depending ownership
- platforms -> Social channels/platforms OR Comms channels depending ownership
- accounts -> Social accounts OR Comms contact profiles depending ownership

Operational Surfaces
- booking/service lifecycle -> Jobs Mode
- inbox/replies/notifications -> Comms Mode
- invoice/payment/reminder -> Finance Mode
- roles/settings/extensions/audit -> Admin Mode

## Rule
Mode ownership must be proven from actual code behavior, not guessed from names.
