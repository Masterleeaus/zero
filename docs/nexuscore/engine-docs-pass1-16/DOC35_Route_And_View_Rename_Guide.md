# DOC 35 — Route and View Rename Guide

Rename order:
1. confirm mode ownership
2. lock term mapping
3. rename folders/files
4. rewrite Blade references
5. rewrite controller view() references
6. rewrite route names
7. rewrite translation keys
8. remove duplicate trees

Preferred route style:
- titan.jobs.*
- titan.comms.*
- titan.finance.*
- titan.admin.*
- titan.social.*

Preferred view path style:
- panel.user.nexus.jobs.*
- panel.user.nexus.comms.*
- panel.user.nexus.finance.*
- panel.user.nexus.admin.*
- panel.user.nexus.social.*
