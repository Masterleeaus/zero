# DOC 30 — Legacy to Mode Label Map

Examples of legacy social-media terminology and their mode-specific reinterpretations.

| Legacy Term | Jobs Mode | Comms Mode | Finance Mode | Admin Mode | Social Mode |
|-------------|-----------|------------|--------------|------------|-------------|
| Post | Job Draft | Message Draft | Financial Draft | Operational Record | Post Draft |
| Campaign | Service Program | Outreach Sequence | Billing Cycle | Compliance Cycle | Campaign |
| Platform | Service Context | Channel | Billing Context | Department Context | Platform |
| Account | Contact Profile | Contact Profile | Ledger Profile | Department Profile | Account |
| Publish | Dispatch / Execute | Deliver | Issue / Post | Audit / Apply | Publish |
| Analytics | Completion Record | Response Record | Ledger Outcome | Status Update | Analytics Record |

Notes:
- These are vocabulary overlays, not permission grants.
- Final locked mappings must follow mode ownership and actual file behavior.
