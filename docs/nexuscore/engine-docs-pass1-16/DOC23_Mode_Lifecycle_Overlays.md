# DOC 23 — Mode Lifecycle Overlays

Universal lifecycle stays the same.
Modes only swap entity language.

Jobs:
Draft → Scheduled Job → Dispatch → Completion → Outcome

Comms:
Message Draft → Send Queue → Delivery → Response → Outcome

Finance:
Financial Draft → Approval → Transaction → Ledger Entry → Outcome

Admin:
Operational Record → Review → Audit → Status Update → Outcome

Social:
Post Draft → Schedule → Publish → Analytics Record → Outcome
