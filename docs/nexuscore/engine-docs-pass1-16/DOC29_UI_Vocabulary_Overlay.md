# DOC 29 — UI Vocabulary Overlay Matrix

Purpose:
Define the approved label overlays for each of the five Nexus modes while preserving the shared chassis.

Modes:
- Jobs
- Comms
- Finance
- Admin
- Social Media

Rules:
1. Labels are mode-owned, not globally reused without review.
2. Same structural slot may receive different visible labels by mode.
3. Shared infrastructure terms stay stable where possible:
   - Draft
   - Schedule
   - Event
   - Outcome
   - Signal
4. Overlays must not blur mode meaning.
