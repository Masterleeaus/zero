# Extraction Decision Rules

## Preserve
Use when the extension surface already matches its final mode and should remain largely intact.

## Repurpose
Use when the surface structure is useful but terminology and ownership must be converted.

## Split
Use when one source surface mixes more than one mode and must be separated safely.

## Merge
Use when multiple source surfaces become one canonical Nexus surface.

## Retire
Use when the source surface is redundant after migration and can be archived.

## Human Decision Required
Use when semantics are ambiguous or business meaning is still unresolved.

## Core Rule
Prefer:
reuse -> repurpose -> split -> merge -> retire
Avoid rebuilding from scratch unless unavoidable.
