# DOC 27 — Signal Promotion Model

Flow:

ProcessRecord
→ Signal emission
→ Validation
→ Consensus / governance
→ Promotion to canonical event
→ Handler execution
→ Outcome record
→ Audit trail

Signal ownership inherits mode ownership.

Examples:
- jobs.job_check_in
- comms.message_sent
- finance.invoice_issued
- admin.policy_reviewed
- social.post_published
